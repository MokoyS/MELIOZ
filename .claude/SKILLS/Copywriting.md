# SKILL — Copywriting & Brand Voice

## Objectif
Écrire des textes qui convertissent sans bullshit.
Chaque mot a une raison d'être. Chaque phrase sert un objectif précis.

---

## Les 4 règles du bon copy web

### 1. Bénéfice avant feature
```
❌ "Notre stack utilise Next.js et Supabase"
✅ "Votre site charge en moins d'une seconde. Toujours."

❌ "Nous proposons un portail Notion dédié"
✅ "Vous suivez l'avancement de votre projet en temps réel. Sans relancer personne."
```

### 2. Précision > généralité
```
❌ "Nous livrons rapidement"
✅ "Livraison en 2 à 4 semaines. Pas 4 mois."

❌ "Notre équipe est disponible"
✅ "Réponse sous 24h. Directement avec celui qui code."

❌ "Des résultats mesurables"
✅ "Score Lighthouse > 90. Visible dans Google en moins de 30 jours."
```

### 3. Montrer, ne pas dire
```
❌ "Nous sommes passionnés"          → dit par tout le monde
❌ "Une équipe créative et dynamique" → vide de sens
❌ "Experts en digital"              → personne ne le croit

✅ Montrer un projet livré en 3 semaines
✅ Montrer le score Lighthouse de 96/100
✅ Montrer le portail Notion du client
```

### 4. Une idée par phrase
```
❌ "MELIOZ crée des sites web modernes, performants et optimisés pour le SEO
    avec une approche centrée sur l'expérience utilisateur."

✅ "MELIOZ crée des sites qui performent.
    Rapides. Bien référencés. Pensés pour l'utilisateur."
```

## Structure d'une page qui convertit

```
1. HERO — Répondre à "Pourquoi vous ?"
   → Titre percutant + sous-titre précis + CTA visible

2. SOCIAL PROOF — Prouver que ça marche
   → Chiffres, logos clients, témoignages, réalisations

3. SERVICES — Répondre à "Qu'est-ce que vous faites ?"
   → Bénéfice en titre, détails en corps, tag technique en bas

4. PROCESSUS — Répondre à "Comment ça se passe ?"
   → Rassurer, éliminer la peur de l'inconnu

5. DIFFÉRENCIATEUR — Répondre à "Pourquoi vous plutôt qu'un autre ?"
   → Le manifeste, l'approche unique, ce qui ne se trouve pas ailleurs

6. CTA FINAL — Provoquer l'action
   → Une seule action possible, friction minimale
```

## Formules copywriting éprouvées

### Titre hero
```
[Résultat désirable] + [sans la douleur habituelle]
→ "Un site qui performe. Sans attendre 6 mois."

[Problème connu] → [Solution MELIOZ]
→ "Votre agence ne répond plus ? MELIOZ livre en 3 semaines."

[Ce que le client veut vraiment]
→ "Votre projet mérite une équipe qui le comprend."
```

### Description de service
```
Ligne 1 : Le bénéfice principal (ce que le client gagne)
Ligne 2 : Ce que ça inclut concrètement (3 éléments max)
Ligne 3 : La preuve ou le chiffre différenciant
Tag : La stack / techno (pour les clients tech-aware)
```

### CTA
```
❌ "Contactez-nous"    → passif, générique
❌ "En savoir plus"    → ne dit rien
❌ "Cliquez ici"       → ne dit rien

✅ "Démarrer un projet" → action claire
✅ "Parler de mon projet" → personnalisé
✅ "Voir les réalisations" → promesse de preuve
✅ "Obtenir un devis"   → intention claire
```

## Règles de style MELIOZ

```
VOIX     : MELIOZ (troisième personne sur le site)
REGISTRE : Professionnel mais direct. Pas de jargon. Pas de langue de bois.
PONCTUATION : Phrases courtes. Points finaux sur les titres.
MAJUSCULES : Labels en UPPERCASE. Titres en Title case.
CHIFFRES : Toujours en chiffres (3 semaines, pas "trois semaines")
ÉVITER  : "passionné", "innovant", "disruptif", "synergies", "sur mesure"
         (sur mesure est ok si justifié, éviter en accroche)
```

## Hiérarchie de l'information

```
Titre H1  → La promesse principale. 1 idée. Max 8 mots.
Sous-titre → Le contexte. 1-2 phrases. Max 20 mots.
Corps      → Les détails. 2-3 phrases. 1 idée par paragraphe.
CTA        → L'action. 2-4 mots. Verbe d'action.
```
