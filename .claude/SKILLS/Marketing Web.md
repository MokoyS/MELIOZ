# SKILL — Marketing Web & Conversion

## Objectif
Structurer chaque page pour maximiser la conversion.
Un visiteur doit comprendre en 5 secondes pourquoi MELIOZ et pas un autre.

---

## Les 5 objections à éliminer sur un site d'agence

```
1. "Est-ce qu'ils sont compétents ?"
   → Réponses : réalisations, stack visible, scores techniques, formations

2. "Est-ce qu'ils vont respecter mon budget ?"
   → Réponses : tarifs affichés, dévis fixe mentionné, acompte 30% expliqué

3. "Est-ce qu'ils vont respecter les délais ?"
   → Réponses : délais précis par offre, processus en étapes, portail de suivi

4. "Est-ce que je vais comprendre ce qui se passe ?"
   → Réponses : portail Notion, points hebdomadaires, Loom de livraison

5. "Que se passe-t-il après la livraison ?"
   → Réponses : recette 15 jours, abonnements Run, transfert des accès
```

## Sections à avoir absolument sur le site MELIOZ

### Page d'accueil
```
✅ Hero avec proposition de valeur claire
✅ Section "Ce qu'on fait" (3-4 services max)
✅ Section chiffres / preuves sociales
✅ Section manifeste / différenciateur
✅ Section processus (rassurer)
✅ Section offres (réduire la friction prix)
✅ CTA final fort
```

### Page Services
```
✅ Hero de la page avec accroche
✅ Grille de services avec détails
✅ Pour qui c'est adapté (par service)
✅ Ce qui est inclus (liste claire)
✅ Ce qui n'est pas inclus (évite les malentendus)
✅ FAQ du service
✅ CTA vers devis / contact
```

### Page Équipe
```
✅ Photo + prénom + rôle précis
✅ 2-3 lignes de bio orientées compétence (pas passion)
✅ Stack / expertise concrète
✅ Lien LinkedIn ou GitHub
✅ Ce que chacun apporte au projet client
```

---

## Preuves sociales — Par ordre d'efficacité

```
1. Témoignages clients avec nom + prénom + entreprise + photo
2. Réalisations avec résultats mesurables (pas juste des screenshots)
3. Logos clients
4. Chiffres (X projets livrés, X% de satisfaction, X semaines en moyenne)
5. Certifications / formations
```

---

## Sections à ajouter sur agencemelioz.com

### Section "Chiffres" (entre hero et services)
```
Format : 3-4 métriques en grand
- X projets livrés
- X semaines délai moyen
- Score Lighthouse > 90 sur tous nos sites
- Réponse sous 24h garantie
```

### Section "Pour qui ?" (après les services)
```
3 profils cibles avec leurs problèmes spécifiques et notre réponse :
- Indépendants / TPE
- Startups & fondateurs
- PME & ETI
```

### Section FAQ (avant le CTA final)
```
Questions réelles des clients :
- "Combien ça coûte vraiment ?"
- "Combien de temps ça prend ?"
- "Que se passe-t-il si je ne suis pas satisfait ?"
- "Est-ce que je peux modifier le site moi-même après ?"
- "Vous travaillez sur quel type de projet ?"
```

### Section Réalisations (page dédiée ou section homepage)
```
Pour chaque réalisation :
- Nom du client / projet
- Type de projet
- Délai de livraison
- Technologies utilisées
- Résultat concret (trafic, conversion, score)
- Screenshot ou lien live
```
